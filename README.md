# Toasty

Toasty is a youtube clone built with Typescript and Google Cloud Services

## Features

* Sign in with Google
* Upload videos up to 4k resolution
* List videos
* Watch a video

## Tech Stack

* TypeScript
* Vitejs
* Express.js
* Docker
* FFmpeg
* Firebase Authentication
* Firebase Functions
* Firebase Firestore
* Google Cloud Storage
* Google Cloud Pub/Sub
* Google Cloud Run
